# Lost and Found App – SIT708 Task 7.1

##  Project Overview
This is a Lost and Found Android application developed using Java and SQLite. The app allows users to report lost or found items, store data locally, and manage adverts.

##  Features
- Create Lost/Found advert
- Store data using SQLite database
- View all adverts using RecyclerView
- Search/filter by category
- Upload image for each item
- Automatic timestamp for each post
- Remove advert after item is returned

## Technologies Used
- Android Studio
- Java
- SQLite
- XML Layouts
- RecyclerView

## How to Run
1. Open project in Android Studio
2. Sync Gradle
3. Run on emulator or physical device
